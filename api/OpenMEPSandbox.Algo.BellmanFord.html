GetShortestPathAndDistance(int, int)

Finds the shortest path from the given source vertex to the given destination vertex using the Bellman-Ford algorithm, and returns an array of the vertices along the path in the order they are visited, as well as the total distance of the path. If there is no path from the source to the destination, or if there is a negative weight cycle in the graph, returns null.
